🤖 Antigravity / Stitch Output Settings
✅ These settings tell Antigravity exactly how to structure the design so every element is fully editable in Elementor editor — zero HTML code needed after import.
Unique Container IDsEvery Section/Column gets a unique CSS ID (e.g., #section-hero-001) — Elementor Advanced tab mein visible, easy to target
Unique CSS Classes per ElementEvery widget/element gets its own class (e.g., .btn-cta-hero, .card-service-1) — no shared classes that conflict
Named Sections / ElementsEvery section/widget labeled with descriptive name in Elementor panel (e.g., "Hero Section", "Services Grid") — easy to navigate
No Inline Styles — Use Elementor Style Tab OnlyAll styling via Elementor Style/Advanced tabs, not hardcoded inline CSS — fully editable without code
Global Colors as CSS VariablesPrimary & secondary colors defined as CSS variables and Elementor Global Colors — change once, updates everywhere
Responsive Breakpoints SetMobile (375px), Tablet (768px), Desktop (1280px) — all sections must be tested responsive in Elementor editor
WhatsApp Button via Plugin (not hardcoded)Use selected WhatsApp plugin instead of coding — so client can change number themselves from WP dashboard
Images as Elementor Dynamic / PlaceholderAll images use Elementor image widget placeholders — client can change from editor without touching code
